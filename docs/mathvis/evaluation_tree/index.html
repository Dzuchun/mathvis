Module mathvis::evaluation_tree

source ·
Expand description

This module defines Evaluatable trait as well as types implementing it that are intended to be used.

EvaluationTree is a go-to type to use during your evaluations. Strictly-speaking, it can be anything, and might even not contain actual Nodes (For now it does contains a single root node though).

EvaluationTree can be created by parsing with EvaluationTree::from_tokens function. If you need some custom structure, EvaluationTree can be ignored and bare Nodes used instead, as they both implement Evaluatable anyway.
